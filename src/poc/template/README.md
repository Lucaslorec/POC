# POC Template

This directory contains templates for creating new POCs.

## How to Create a New POC

1. Create a new directory under `src/poc/` for your POC
2. Copy these template files to your new directory
3. Customize the files for your specific POC

## Files to Create

1. `.env.sample` - Template showing required environment variables
2. `config.ts` - TypeScript module that loads and validates environment variables
3. `index.ts` - Main implementation of your POC
4. `README.md` - Documentation for your POC
